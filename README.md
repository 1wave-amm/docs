# 1wave Docs Repo

This repository contains the **1wave product documentation**.

## Structure

- `docs/`: the documentation content (GitBook-ready and compatible with GitHub Pages).

## Publishing

- **GitBook**: connect the repo and set the project directory to `docs/`. Use `docs/SUMMARY.md` as the table of contents.
- **GitHub Pages**: enable Pages on `main` using the `/docs` folder.

# Glossary

- **Aqua**: protocol di 1inch per strategie/AMM “app-based”.
- **AquaApp**: smart contract applicativo che esegue operazioni su Aqua (qui: `WaveSwap`).
- **Maker**: address che fornisce liquidità/strategie ad Aqua (qui: `AquaAdapter`).
- **Pair**: coppia token + fee (BPS) configurata sull’adapter.
- **Strategy**: configurazione (maker, token0, token1, feeBps, salt) usata da Aqua per gestire bilanci.
- **StudioProFactory**: factory di FactorDAO per creare vault Pro.
- **StudioProVault**: vault Pro creato dalla factory; emette eventi deposit/withdraw ecc.
- **Subgraph**: indicizzatore GraphQL di eventi on-chain.
- **Merkl**: infrastruttura per point programs e calcolo reward off-chain con token “point”.

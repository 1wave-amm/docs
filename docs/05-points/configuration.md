# Configuration

File: `points-service/.env`

## Server

- `PORT` (default `3001`)
- `NODE_ENV`

## Subgraph

- `SUBGRAPH_URL` (default: Goldsky public endpoint)

## Merkl

- `MERKL_API_URL` (default `https://api.merkl.xyz/v4`)
- `MERKL_API_KEY`
- `MERKL_CREATOR_ADDRESS`
- `MERKL_DISTRIBUTOR_ADDRESS`

## Blockchain

- `CHAIN_ID` (default `8453`)
- `RPC_URL` (default `https://mainnet.base.org`)

## Token addresses

- `POINTS_TOKEN_ADDRESS`
- `TOKEN_1INCH_ADDRESS`
- `TOKEN_FCTR_ADDRESS`

## Rates (points calculation)

- `POINTS_PER_USD_TVL`
- `POINTS_PER_USD_DEPOSIT_PER_DAY`
- `POINTS_PER_VAULT_OWNED`
- `REFERRAL_BONUS_MULTIPLIER`
- `TOKEN_BOOST_MULTIPLIER`

# 1wave Documentation

Benvenuto nella documentazione di **1wave**, un ecosistema di:

- AMM basato su **1inch Aqua** (`WaveSwap`)
- Adapter per la gestione delle strategie (`AquaAdapter`)
- Sistema di **points non-transferable** integrato con **Merkl** (`WavePointToken` + Points Service)

- **[Points Service & API](points-service.md)**: servizio backend e endpoint HTTP.
- **[Deployments](deployments.md)**: indirizzi su Base e link a explorer.
